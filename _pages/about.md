Based in **Tiznit, Morocco**, I operate at the intersection of software engineering, machine learning research, and network infrastructure design. My work ranges from publishing Android tools with 100,000+ downloads to training deep neural networks and managing local wireless internet service provider (WISP) backbones.

---

## 🚀 Core Competencies

### 💻 Mobile & Game Development
*   **Android App Publishing:** Architect and maintain high-performance utility and security applications on the Google Play Store (e.g., *Internet Speed Meter*, *Calculator Vault*), focusing on memory efficiency, security protocols, and encryption schemas.
*   **Web Games:** Develop lightweight, canvas-rendered instant games for social cross-sharing (e.g., *Solitaire Classic* for Facebook Instant Games).

### 🧠 Machine Learning & Deep Learning
*   **Applied AI Research:** Implement predictive and generative models across various domains, including:
    *   *Natural Language Processing:* Question duplicate detection (Quora similarity) and clinical text mutation classifiers (personalized cancer diagnosis).
    *   *Graph Mining:* Social network link predictions and friend recommendation configurations.
    *   *Spatio-Temporal Modeling:* Spatio-temporal forecasting regressions (NYC taxi demand).
    *   *Generative Networks:* Melodic chord-sequence modeling (music composition using RNNs).
    *   *Computer Vision:* Behavioral cloning lane-guidance simulators (self-driving cars) and motion classification (HAR accelerometer time-series).

### 🌐 Network & Systems Engineering
*   **WISP Deployments:** Architect and manage a localized community broadband network in Tiznit, using high-gain outdoor sector arrays (TP-Link Pharos CPE series) and core traffic engineering configurations inside MikroTik RouterOS.
*   **Virtualization & Servers:** Maintain a self-hosted home virtualization node containerizing utility applications using Docker, Docker Compose, and Nginx reverse proxies.
